# Бренд
Путь `permission.yml > server.brand`

## Пояснение
Категория, отвечающая за настройку прав у бренда сервера

## Редактирование
```yaml
<permission.server.brand>
```

### По умолчанию
```yaml
brand:
  name: "flectonepulse.module.server.brand"
  type: TRUE
```
