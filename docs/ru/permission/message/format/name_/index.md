# Имя
Путь `permission.yml > message.format.name`

## Пояснение
Категория, отвечающая за настройку прав у имени игрока

## Редактирование
```yaml
<permission.message.format.name>
```

### По умолчанию
```yaml
name-:
  name: "flectonepulse.module.message.format.name"
  type: TRUE
```
