# Иконка
Путь `permission.yml > message.status.icon`

## Пояснение
Категория, отвечающая за настройку прав у иконки сервера

## Редактирование
```yaml
<permission.message.status.icon>
```

### По умолчанию
```yaml
icon:
  name: "flectonepulse.module.message.status.icon"
  type: TRUE
```
