# В табе
Путь `permission.yml > message.objective.tabname`

## Пояснение
Категория, отвечающая за настройку прав у значений игрока в списке

## Редактирование
```yaml
<permission.message.objective.tabname>
```

### По умолчанию
```yaml
messagelist:
  name: "flectonepulse.module.message.objective.tabname"
  type: TRUE
```
