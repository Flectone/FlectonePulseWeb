# Сообщения
Путь `permission.yml > message.anvil`

## Пояснение
Категория, отвечающая за настройку прав у сообщений игрока в наковальне

## Редактирование
```yaml
<permission.message.anvil>
```

### По умолчанию
```yaml
anvil:
  name: "flectonepulse.module.message.anvil"
  type: TRUE
```
