# Упоминание
Путь `localizations > en_us.yml > message.format.mention`

## Пояснение
Форматирование упоминания

## Редактирование
```yaml
<en_us.message.format.mention>
```

### По умолчанию
```yaml
mention:
  format: "<fcolor:2>@<player></fcolor>"
```
