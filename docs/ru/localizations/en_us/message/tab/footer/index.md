# Нижнее
Путь `localizations > en_us.yml > message.tab.footer`

## Пояснение
Сообщения, отвечающие за нижнюю часть ТАБа
![footer](/footer.png)

## Редактирование
```yaml
<en_us.message.tab.footer>
```

### По умолчанию
```yaml
footer:
  values:
    - "<br><fcolor:1>TPS <tps>, Online <online><br>"
    - "<br><fcolor:1>Hello <fcolor:2><player></fcolor:2>!<br>"
```
