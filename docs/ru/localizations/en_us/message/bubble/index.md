# Над головой
Путь `localizations > en_us.yml > message.bubble`

## Пояснение
Сообщения над головой при использовании чата игроком
![bubble](/bubble.gif)

## Редактирование
```yaml
<en_us.message.bubble>
```

### По умолчанию
```yaml
bubble:
  format: "<fcolor:3><message>"
```
