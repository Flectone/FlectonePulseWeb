# Автоматическое
Путь `localizations > ru_ru.yml > message.auto`

## Пояснение
Автоматическое сообщение раз в какой-то промежуток времени
![auto](/auto.png)

## Редактирование
```yaml
<ru_ru.message.auto>
```
