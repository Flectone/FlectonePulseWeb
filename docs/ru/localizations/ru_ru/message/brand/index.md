# Бренд
Путь `localizations > ru_ru.yml > message.brand`

## Пояснение
Сообщения бренда сервера
![brand](/brand.png)

## Редактирование
```yaml
<ru_ru.message.brand>
```

### По умолчанию
```yaml
brand:
  values:
    - "<white>Майнкрафт"
    - "<aqua>Майнкрафт"
```

## Параметры

- [Сообщения](/ru/message/brand/)
- [Права](/ru/permission/message/brand/)

### `values`
- По умолчанию:
```yaml
- "<white>Майнкрафт"
- "<aqua>Майнкрафт"
```
