# Время
Путь `messages > ru_ru.yml > time`

## Пояснение
Категория, отвечающее за форматирование времени

## Редактирование
```yaml
<ru_ru.time>
```

### По умолчанию
```yaml
time:
  day: "д"
  hour: "ч"
  minute: "м"
  second: "с"
```

## Параметры

### `day`
- По умолчанию `д`

Время, отвечающее за день

### `hour`
- По умолчанию `ч`

Время, отвечающее за час

### `minute`
- По умолчанию `м`

Время, отвечающее за минуту

### `second`
- По умолчанию `с`

Время, отвечающее за секунду
