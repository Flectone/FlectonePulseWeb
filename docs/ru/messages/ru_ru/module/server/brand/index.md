# Бренд
Путь `messages > ru_ru.yml > module.server.brand`

## Пояснение
Сообщения бренда сервера
![brand](/brand.png)

## Редактирование
```yaml
<ru_ru.module.server.brand>
```

### По умолчанию
```yaml
brand:
  values:
    - "<white>Майнкрафт"
    - "<aqua>Майнкрафт"
```

## Параметры

- [Конфиг](/ru/config/module/server/brand/)
- [Права](/ru/permissions/module/server/brand/)

### `values`
- По умолчанию:
```yaml

```
