# Приветствие
Путь `messages > ru_ru.yml > module.server.message.greeting`

## Пояснение
Личное сообщение игроку при подключении на сервер
![greeting](/greeting.png)

## Редактирование
```yaml
<ru_ru.module.server.message.greeting>
```

### По умолчанию
```yaml
greeting:
  format:
    - " "
    - "[#][#][#][#][#][#][#][#]"
    - "[#][#][#][#][#][#][#][#]"
    - "[#][#][#][#][#][#][#][#]"
    - "[#][#][#][#][#][#][#][#]"
    - "[#][#][#][#][#][#][#][#]  <fcolor:1>Привет,"
    - "[#][#][#][#][#][#][#][#]  <player>"
    - "[#][#][#][#][#][#][#][#]"
    - "[#][#][#][#][#][#][#][#]"
    - " "
```
