# Комманда /do
Путь `messages > en_us.yml > module.command.do`

## Пояснение
Сообщения для комманды `/do`
![command do](/commanddo.png)

## Редактирование
```yaml
<en_us.module.command.do>
```

### По умолчанию
```yaml
do:
  format: "<fcolor:1>✎ <message> (<i><display_name></i>)"
```
