## Виды прав
| Вид      | Объяснение                                                   |
|----------|--------------------------------------------------------------|
| `TRUE`   | Доступно для любого игрока                                   |
| `FALSE`  | Доступно если выдано отдельно (например с помощью LuckPerms) |
| `OP`     | Доступно для операторов сервера                              |
| `NOT_OP` | Доступно если выдано отдельно (например с помощью LuckPerms) |
