# Табличка
Путь `message.yml > sign`

## Пояснение
Модуль, отвечающий за сообщения на табличке
![sign](/signoak.gif)

## Редактирование
```yaml
<message.sign>
```

### По умолчанию
```yaml
sign:
  enable: true
  sound:
    enable: false
    type: "BLOCK_NOTE_BLOCK_BELL:1:1"
```

## Параметры

- [Права](/ru/permission/message/sign/)

### `enable`
- По умолчанию `true`

Включает или выключает работоспособность модуля
