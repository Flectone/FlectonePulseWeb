# Контактирование
Путь `message.yml > contact.*`

## Пояснение
Модуль, отвечающий за сообщения при взаимодействии

## Редактирование
```yaml
<message.contact>
```

### По умолчанию
```yaml
contact:
  enable: true
```

## Параметры

- [Права](/ru/permission/message/contact/)

### `enable`
- По умолчанию `true`

Включает или выключает работоспособность модуля и его наследников
