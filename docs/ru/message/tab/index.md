# Таб
Путь `message.yml > module.server.tab.*`

## Пояснение
Модуль, отвечающий за настройку ТАБа
![tab](/tab.png)

## Редактирование
```yaml
<message.module.server.tab>
```

### По умолчанию
```yaml
tab:
  enable: true
```

## Параметры

- [Права](/ru/permission/message/tab/)

### `enable`
- По умолчанию `true`

Включает или выключает работоспособность модуля и его наследников
