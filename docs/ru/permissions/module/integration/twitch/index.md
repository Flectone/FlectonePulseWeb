# Твич
Путь `permissions.yml > module.integration.twitch`

## Пояснение
Категория, отвечающая за настройку прав у интеграции Twitch

## Редактирование
```yaml
<permissions.module.integration.twitch>
```

### По умолчанию
```yaml
twitch:
  name: "flectonepulse.module.integration.twitch"
  type: TRUE
```
