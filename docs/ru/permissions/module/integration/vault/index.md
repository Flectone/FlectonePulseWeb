# Vault
Путь `permissions.yml > module.integration.vault`

## Пояснение
Категория, отвечающая за настройку прав у интеграции Vault

## Редактирование
```yaml
<permissions.module.integration.vault>
```

### По умолчанию
```yaml
vault:
  name: "flectonepulse.module.integration.vault"
  type: TRUE
```
