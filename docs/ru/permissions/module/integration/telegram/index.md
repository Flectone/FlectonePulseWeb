# Телеграм
Путь `permissions.yml > module.integration.telegram`

## Пояснение
Категория, отвечающая за настройку прав у интеграции Telegram

## Редактирование
```yaml
<permissions.module.integration.telegram>
```

### По умолчанию
```yaml
telegram:
  name: "flectonepulse.module.integration.telegram"
  type: TRUE
```
