# Версия
Путь `permissions.yml > module.server.status.version`

## Пояснение
Категория, отвечающая за настройку прав у версии сервера

## Редактирование
```yaml
<permissions.module.server.status.version>
```

### По умолчанию
```yaml
version:
  name: "flectonepulse.module.server.status.version"
  type: TRUE
```
