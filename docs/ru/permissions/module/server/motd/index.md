# МОТД
Путь `permissions.yml > module.server.motd`

## Пояснение
Категория, отвечающая за настройку прав у МОТД сервера

## Редактирование
```yaml
<permissions.module.server.motd>
```

### По умолчанию
```yaml
motd:
  name: "flectonepulse.module.server.motd"
  type: TRUE
```
