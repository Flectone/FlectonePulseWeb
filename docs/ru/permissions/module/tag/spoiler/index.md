# Спойлер
Путь `permissions.yml > module.tag.spoiler`

## Пояснение
Категория, отвечающая за настройку прав у тегов спойлеров

## Редактирование
```yaml
<permissions.module.tag.spoiler>
```

### По умолчанию
```yaml
spoiler:
  name: "flectonepulse.module.tag.spoiler"
  type: TRUE
```
