# Афк
Путь `permissions.yml > module.player.afk`

## Пояснение
Категория, отвечающая за настройку прав у игрока Афк

## Редактирование
```yaml
<permissions.module.player.afk>
```

### По умолчанию
```yaml
afk:
  name: "flectonepulse.module.player.afk"
  type: TRUE
```
