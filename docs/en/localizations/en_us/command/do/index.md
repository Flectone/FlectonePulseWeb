# Комманда /do
Путь `localizations > en_us.yml > command.do`

## Пояснение
Сообщения для комманды `/do`
![command do](/commanddo.png)

## Редактирование
```yaml
<en_us.command.do>
```

### По умолчанию
```yaml
do:
  format: "<fcolor:1>✎ <message> (<i><display_name></i>)"
```
