# Смайлики
Путь `permission.yml > message.format.emoji`

## Пояснение
Категория, отвечающая за настройку прав у тегов смайликов

## Редактирование
```yaml
<permission.message.format.emoji>
```

### По умолчанию
```yaml
emoji:
  name: "flectonepulse.module.message.format.emoji"
  type: TRUE
```
