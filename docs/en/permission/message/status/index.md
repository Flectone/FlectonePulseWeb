# Статус
Путь `permission.yml > message.status.*`

## Пояснение
Категория, отвечающая за настройку прав у просмотра сервера

## Редактирование
```yaml
<permission.message.status>
```

### По умолчанию
```yaml
status:
  name: "flectonepulse.module.message.status"
  type: TRUE
```
