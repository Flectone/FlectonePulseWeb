# Твич
Путь `permission.yml > integration.twitch`

## Пояснение
Категория, отвечающая за настройку прав у интеграции Twitch

## Редактирование
```yaml
<permission.integration.twitch>
```

### По умолчанию
```yaml
twitch:
  name: "flectonepulse.module.integration.twitch"
  type: TRUE
```
