# SuperVanish
Путь `permission.yml > integration.supervanish`

## Пояснение
Категория, отвечающая за настройку прав у интеграции SuperVanish

## Редактирование
```yaml
<permission.integration.supervanish>
```

### По умолчанию
```yaml
supervanish:
  name: "flectonepulse.module.integration.supervanish"
  type: TRUE
```
