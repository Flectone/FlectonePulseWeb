# Теги
Путь `permissions.yml > module.tag.*`

## Пояснение
Категория, отвечающая за настройку прав у тегов

## Редактирование
```yaml
<permissions.module.tag>
```

### По умолчанию
```yaml
server:
  name: "flectonepulse.module.tag"
  type: TRUE
```

## Параметры

- [Конфиг](/en/config/module/tag/)

### `name`
- По умолчанию `flectonepulse.module.tag`

Название права

### `type`
- По умолчанию `TRUE`

Тип права
