# Бренд
Путь `permissions.yml > module.server.brand`

## Пояснение
Категория, отвечающая за настройку прав у бренда сервера

## Редактирование
```yaml
<permissions.module.server.brand>
```

### По умолчанию
```yaml
brand:
  name: "flectonepulse.module.server.brand"
  type: TRUE
```
