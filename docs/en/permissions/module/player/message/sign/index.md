# Табличка

## Пояснение
Категория, отвечающая за настройку прав у сообщений игрока в табличке

## Редактирование
```yaml
<permissions.module.player.message.sign>
```

### По умолчанию
```yaml
sign:
  name: "flectonepulse.module.player.message.sign"
  type: TRUE
```
