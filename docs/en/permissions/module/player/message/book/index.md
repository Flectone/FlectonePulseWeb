# Книга
Путь `permissions.yml > module.player.message.book`

## Пояснение
Категория, отвечающая за настройку прав у сообщений игрока в книге

## Редактирование
```yaml
<permissions.module.player.message.book>
```

### По умолчанию
```yaml
book:
  name: "flectonepulse.module.player.message.book"
  type: TRUE
```
