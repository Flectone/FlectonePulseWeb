# Команда
Путь `permissions.yml > module.player.team`

## Пояснение
Категория, отвечающая за настройку прав у команды игрока

## Редактирование
```yaml
<permissions.module.player.team>
```

### По умолчанию
```yaml
team:
  name: "flectonepulse.module.player.team"
  type: TRUE
```
