# Имя
Путь `permissions.yml > module.player.name`

## Пояснение
Категория, отвечающая за настройку прав у имени игрока

## Редактирование
```yaml
<permissions.module.player.name>
```

### По умолчанию
```yaml
name-:
  name: "flectonepulse.module.player.name"
  type: TRUE
```
