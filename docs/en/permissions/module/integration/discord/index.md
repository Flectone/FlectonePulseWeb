# Дискорд
Путь `permissions.yml > module.integration.discord`

## Пояснение
Категория, отвечающая за настройку прав у интеграции Discord

## Редактирование
```yaml
<permissions.module.integration.discord>
```

### По умолчанию
```yaml
discord:
  name: "flectonepulse.module.integration.discord"
  type: TRUE
```
