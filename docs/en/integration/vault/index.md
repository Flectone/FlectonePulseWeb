# Vault
Путь `integration.yml >vault`

## Пояснение
Интеграция с Vault позволяет проверять права у **оффлайн игроков**

## Редактирование
```yaml
<integration.vault>
```

### По умолчанию
```yaml
vault:
  enable: true
```

## Параметры

- [Права](/ru/permission/integration/vault/)

### `enable`
- По умолчанию `true`

Включает или выключает работоспособность модуля
