# Times
Path `messages > en_us.yml > time`

## Explanation
Category for time formatting

## Edit
```yaml
<en_us.time>
```

### Default
```yaml
time:
  day: "d"
  hour: "h"
  minute: "m"
  second: "s"
```

## Options

### `day`
- Default `d`

Time for day

### `hour`
- Default `h`

Time for hour

### `minute`
- Default `m`

Time for minute

### `second`
- Default `s`

Time for second
