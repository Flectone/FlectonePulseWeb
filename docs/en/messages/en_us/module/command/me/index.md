# Комманда /me
Путь `messages > en_us.yml > module.command.me`

## Пояснение
Сообщения для комманды `/me`
![command me](/commandme.png)

## Редактирование
```yaml
<en_us.module.command.me>
```

### По умолчанию
```yaml
me:
  format: "<fcolor:1>✎ <display_name> <message>"
```
