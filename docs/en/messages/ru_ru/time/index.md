# Times
Path `messages > ru_ru.yml > time`

## Explanation
Category for time formatting

## Edit
```yaml
<ru_ru.time>
```

### Default
```yaml
time:
  day: "д"
  hour: "ч"
  minute: "м"
  second: "с"
```

## Options

### `day`
- Default `д`

Time for day

### `hour`
- Default `ч`

Time for hour

### `minute`
- Default `м`

Time for minute

### `second`
- Default `с`

Time for second
