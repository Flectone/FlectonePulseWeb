# Brand

## Explanation
Server brand messages
![brand](/brand.png)

## Edit
```yaml
<ru_ru.module.server.brand>
```

```yaml
brand:
  format:
    - "<white>FlectonePulse"
    - "<aqua>FlectonePulse"
```

